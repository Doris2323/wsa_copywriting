# 水球軟體學院 — AI Marketing 文案工作區

## 主要工作事項

本工作區負責水球軟體學院的行銷文案相關作業，包含風格研究、社群文案撰寫、關鍵字分析與推廣素材產出。

### 1. 分析及撰寫規則
- 水球潘寫作風格研究（用字、口吻、修辭、文章結構、說服策略）
- 社群文案寫作規範與去除 AI 味 SOP
- 四大主題框架（社群凝聚力、軟工基礎、工程師價值提升、工程師轉型）
- Threads 貼文撰寫完整執行規則（一開文 + 二串文）

### 2. Threads 關鍵字排行榜分析
- 每兩週執行一次，追蹤 15 個關鍵字在 Threads 上的討論度
- 涵蓋技術工具、AI 概念、軟體工程方法論、品牌監測四大類
- 產出排行榜、深度分析、行銷策略建議與競品觀察

### 3. 社群文案撰寫
- 依據水球潘寫作風格與最新關鍵字報告，產出 Threads / Facebook 社群文案
- 針對不同平台與受眾量身調整切入角度與語言調性

## 執行流程

### Threads 貼文產出流程

```
1. 執行關鍵字分析（每兩週）
   → 依照 05_Thread熱門關鍵字排行榜分析規則.md
   → 產出至 關鍵字排行榜分析/Threads關鍵字排行榜分析報告_{日期}.md

2. 撰寫 Threads 貼文（每週）
   → 依照 01_Threads社群貼文撰寫.md
   → 自動讀取 02、03、04 規範 + 最新關鍵字報告
   → 產出至 文案撰寫/WSA Threads 社群貼文_{日期}.md
```

### 快速指令

| 任務 | 指令 |
|------|------|
| 關鍵字分析 | 「請依照 `05_Thread熱門關鍵字排行榜分析規則.md` 執行本期分析」 |
| 撰寫貼文（指定主題） | 「請依照 `01_Threads社群貼文撰寫.md` 撰寫一則主題二的貼文」 |
| 撰寫貼文（一週份） | 「請依照 `01_Threads社群貼文撰寫.md` 撰寫本週 5 則貼文」 |
| 蹭熱度貼文 | 「請讀取最新關鍵字報告，撰寫 3 則蹭熱度貼文」 |
