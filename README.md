# 水球軟體學院 — AI Marketing 文案工作區

## 主要工作事項

本工作區負責水球軟體學院的行銷文案相關作業，包含風格研究、社群文案撰寫與推廣素材產出。

### 1. 寫作風格分析
- 研究水球潘的寫作風格（用字、口吻、修辭、文章結構、說服策略）
- 分析來源：Threads、官網課程頁、部落格文章、YouTube 頻道
- 產出風格分析報告，作為後續文案撰寫的依據

### 2. 社群文案撰寫
- 依據水球潘寫作風格，產出 Facebook 社團推廣文案
- 針對不同社團受眾量身調整切入角度與語言調性
- 目前推廣目標：AI x BDD 課程問卷填寫邀請
