## 二、各關鍵字深度分析

### 🥇 1. Claude Code — 討論度最高

**Threads 聲量：31M+ threads（標籤累計）**

本週期最熱門話題。Claude Code 創始人 Boris Cherny 持續在 Threads 上分享使用心得與更新，形成強大的社群引力：

- **Claude Code 2.1.0 發布：** 支援 Shift+Enter 換行、Hooks 整合、Skills 前台、萬用字元工具權限等，累計 1,096 commits
- **Claude Code Desktop 新功能：** 可預覽執行中應用、審查程式碼、自動處理 CI 失敗與 PR
- **Boris Cherny 個人使用分享：** 使用 Opus 4.5 搭配 thinking 模式，認為是目前最佳程式編碼模型
- **Spotify 案例：** 最佳開發者自 12 月以來沒寫過一行程式碼，透過 Slack + Claude Code 在通勤時交付 50+ 功能
- **社群討論焦點：** 非程式用途（假期研究、建立投影片、整理信箱、控制烤箱等）

**行銷切入建議：** SDD + Claude Code 的工作流結合是極高潛力話題

---

### 🥈 2. Vibe Coding — 爆發式成長

**Threads 討論量極高，X 平台每月 150K+ 貼文**

Vibe Coding 已成為 2026 年度最火紅的技術關鍵字之一：

- **翻車案例引發大量討論：** 有人 Vibe Code 出 App 後未管好私鑰，帳單超過一萬元，引發安全性大辯論
- **產業分裂討論：** 一派認為 Vibe Coding 是開發民主化，另一派擔憂安全性與品質問題
- **定義共識：** 透過自然語言與 AI Agent 互動進行軟體開發，人類負責指定意圖與評估結果，AI 負責實作

**行銷切入建議：** 「Vibe Coding vs. 軟體工程」的對比討論可創造高互動內容；SDD 可定位為「負責任的 Vibe Coding」

---

### 🥉 3. Cursor — 持續熱門

**市場領導者地位穩固**

- **Cursor 2.5 發布（2026/02/17）：** 推出 Long-running Agents 與 Composer 1.5
- **商業表現：** Series D 募資 $2.3B，年化營收突破 $10 億
- **企業採用案例：** Coinbase（一天完成過去數週的功能）、eBay（從 150 擴展至 500+ 工程師）、Datadog（100% 工程團隊採用）
- **台灣 Threads 討論：** 有人分享 Cursor 多 AI 同步編碼的擴充插件教學

**行銷切入建議：** Cursor + SDD workflow 教學是台灣開發者高關注內容

---

### 4. Agent — 企業級浪潮

- **Anthropic 調查：** 超過半數受訪組織已部署 Agent 處理多階段工作流，80% 回報可衡量的經濟回報
- **Gartner 預測：** 2026 年底前 40% 企業應用將嵌入 AI Agent
- **Agentic AI 佔據 55% 關注度：** 自主規劃、執行、測試、迭代的系統成為主流
- **討論焦點：** MCP vs RAG vs Agent 的架構差異與各自定位

---

### 5. LLM — 基礎設施級話題

**Threads 標籤：123K+ threads**

- 作為基礎技術關鍵字，討論量大但話題已趨於成熟
- 2026 年焦點轉向具體應用（Agent、RAG、MCP）而非 LLM 本身
- 持續有模型比較、最新發布等更新類內容

---

### 6. MCP（Model Context Protocol）— 企業級上升

- **定位清晰化：** MCP = 工具介面（管路）、RAG = 知識注入（記憶）、Agent = 決策迴圈（管理者）
- **SKILL vs MCP 大辯論：** 2026 年重要議題，釐清行為指令 vs 外部工具擴展的差異
- **企業採用：** Snowflake、Canva 等大廠推出 MCP Server
- **進入 CIO 議程：** 企業高層開始關注 MCP 作為 Agent 基礎設施

---

### 7. Prompt Engineering — 微幅降溫

- **轉型趨勢：** 從「Prompt Engineering」轉向「Context Engineering」的討論增加
- **Google 69 頁白皮書：** 涵蓋 zero-shot、few-shot、CoT、ReAct 等技巧，在 Threads 上被大量分享
- **仍列 2026 必備技能第一位：** 但討論深度從「如何學」轉向「如何在生產環境使用」
- **新框架出現：** 推理模型、多模態 prompt、自動最佳化 prompt

---

### 8. GitHub Copilot — 穩定但面臨競爭

- **Copilot SDK 技術預覽版推出**
- **GPT-5.2-Codex 正式上線**
- **新增 Agents Tab 與 Copilot Memory**
- **痛點討論：** Web 版 Coding Agent 啟動需 90+ 秒，用戶體驗待改善
- **市場定位：** 仍是企業首選（特別是有嚴格隱私需求的團隊），但 Cursor 在個人/小團隊市場領先

---

### 9. RAG — 底層技術持續演進

- **三大架構分支：** Pipeline RAG、Agentic RAG、Knowledge Graph RAG
- **核心洞察：** RAG 安靜地驅動大多數現代 AI Agent、Copilot 與聊天系統
- **Agentic RAG 成為實務主流：** 系統不只執行固定的檢索-生成流程，而是自主推理、決策、使用工具

---

### 10. 軟體工程 — 台灣在地討論

- 台灣 Threads 社群中，軟體工程話題多與 AI 工具結合討論
- **熱門方向：** 如何用 AI 完成產品開發文件（規格書、架構圖、Prototype）
- **工具導向：** ChatGPT + Figma 快速生成 Prototype 的實作分享

---

### 11. TDD — 小眾但 AI 時代復興

- **產業共識：** Agile Manifesto 25 週年研討會結論——TDD 從未如此重要
- **AI 最佳搭檔：** TDD 防止 Agent 寫出驗證錯誤行為的測試
- **Jesse Vincent 的 Superpowers 插件：** 將 TDD 直接烘焙進 AI 編碼工作流，已被 Anthropic 官方收入 Marketplace
- **與 Claude Code 整合：** Claude Code 自動執行測試、讀取輸出、修復問題、重新執行

**行銷切入建議：** TDD + AI 是水球軟體學院的核心優勢話題

---

### 12. AI BDD — 極早期萌芽

- **全球討論度極低：** 目前在 Threads 上幾乎沒有專門的 AI BDD 話題
- **但已有產業信號：**
  - BDD 測試工具市場從 2024 年 $1.2 億成長至預估 2033 年 $3 億
  - BDD 2.0（SDD）概念開始出現
  - AI + NLP 生成測試場景的認證訓練已開辦（2026 年 3 月、5 月）
- **水球軟體學院的 SDD 正處於此藍海：** 臺灣規格驅動開發研究組織（SDD.TW）已成立

**行銷切入建議：** 這是目前最大的內容藍海，先佔者優勢明顯

## 三、趨勢洞察與行銷策略建議

### 📊 本期關鍵趨勢

1. **Claude Code 聲量爆發** — 31M+ threads 標籤量，遠超其他 AI 編程工具，且官方社群經營極為成功
2. **Vibe Coding 正反論戰** — 翻車案例引發大量互動，「安全的 Vibe Coding」是內容缺口
3. **TDD + AI 復興** — 產業共識 TDD 是 AI 時代品質保證的關鍵，與 SDD 高度互補
4. **AI BDD / SDD 藍海** — 全球討論度極低但產業需求已顯現，先行者有巨大品牌建立機會
5. **MCP 進入企業主流** — 從技術圈話題升級為 CIO 議程

### 🎯 建議下一期內容主題

| 優先序 | 內容主題 | 搭配關鍵字 | 預期效果 |
|:---:|---------|----------|---------|
| 1 | SDD 是負責任的 Vibe Coding | Vibe Coding, AI BDD, SDD | 蹭熱度 + 定位差異化 |
| 2 | TDD + Claude Code 實戰工作流 | TDD, Claude Code, Agent | 技術含量高、互動潛力大 |
| 3 | MCP + SDD 的企業級開發藍圖 | MCP, Agent, 軟體工程 | 吸引企業受眾 |
| 4 | Cursor vs Claude Code 搭配 SDD 教學 | Cursor, Claude Code, Prompt Engineering | 工具比較類高流量 |
| 5 | 為什麼 AI 時代更需要規格驅動開發 | AI Coding, LLM, RAG | 思想領導力建立 |
